# РЕКОМЕНДАЦИИ ПО ИЗОБРАЖЕНИЯМ
# =====================================

# 1. ФОРМАТ: PNG с прозрачным фоном (transparency)
# 2. КАЧЕСТВО: Высокое разрешение, минимум 256x256 для иконок
# 3. СТИЛЬ: Единый визуальный стиль (плоский дизайн или 3D иллюстрации)
# 4. ЦВЕТА: Яркие, насыщенные цвета, подходящие для детей
# 5. ПРОСТОТА: Четкие, узнаваемые изображения без лишних деталей
# 6. ИСТОЧНИКИ: 
#    - Flaticon.com (бесплатные иконки с attribution)
#    - Freepik.com (качественные векторные изображения)
#    - Icons8.com (большая библиотека иконок)
#    - Unsplash.com / Pexels.com (фотографии, если нужны реалистичные)
#    - Canva.com (создание собственных иконок)
